1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

El uso del !IMPORTANT en CSS debe de ser limitado y cuidadoso. Solo debe de ser usado para anular estilos específicos o garantizar que ciertos estilos prevalezcan, pero se debe de evitar el exceso de uso. Se debe de priorizar la especifidad del selector y la organización del código CSS para mantenerlo claro y mantenible.

2.Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

La elección de una imagen de fondo en HTML se debe de ser cuidadoso debido a que tiene mucho impacto en el diseño y visibilidad de la página, debido a que le legibilidad del texto, contraste visual y la coherencia pueden verser afectados. La imagen debe se ser discreta y no dificultar la comprensión.

3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Cada uno tiene diferentes funciones las cuales son: GET: solicitar datos del servidor, POST: Envia datos al servidor para crear un recurso, HEAD: Solicita solo los encabezados, sin el cuerpo, PUT: Reemplaza o actualiza un recurso en el servidor, PATCH: Modifica parcialmente un recurso, DELETE: Elimina un recurso del servidor.

4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Este ayudará al rendimiento del sitio, debido a que reduce el tamaño del archivo. Lo que resulta como un tiempo de carga más rápido, ya que disminuye el tiempo de transferencia de datos entre el servidor y el navegador.
