1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una inmensa red de computadoras de todo el mundo conectadas entre sí, mientras que la World Wide Web es una enorme colección de páginas que se asienta sobre esa red de computadoras.

2. ¿Cuáles son las partes de una URL?

Protocol, Domain, Path, Parameters.

3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Solicitar datos de un recurso en específico donde los datos no se modifican de ninguna manera.
POST: Utiliza este método para enviar datos a un servidor para crear un recurso.
PUT: Actualizar recursos existentes en un servidor. “Editar” algo.
HEAD: Recuperar encabezados.
PATCH: Método para aplicar modificaciones parciales en un recurso.
DELETE: Eliminar recurso especificado.
